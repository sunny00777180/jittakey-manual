JittaKey 사용설명서 (2025.06.29 기준)

✨ 개요: WYSIWYG 일본어 입력의 혁신

JittaKey는 기존의 로마자 변환 방식을 근본적으로 개선한 직관적(WYSIWYG) 일본어 입력 방식을 제공합니다.

  • 입력한 그대로 표시 (What You See Is What You Get)
  • 로마자 베이스 사용자에게도 빠르게 적응 가능
  • 입력 횟수 약 20% 이상 감소
  • 시각적 연상법을 통한 빠른 학습

🔧 핵심 입력 원리

  • 「시」 = S / 「요」 = L → S + L = 「쇼」
  • 보이는 대로 음절을 순차적으로 입력 → 변환 대기 없이 바로 출력
  • 예: 「쇼」→ s + l 입력 (syo가 아님)
  • 변환 대기 없이 즉시 결과 확인

🈂️ 스테가나 (작은 글자) 입력

  • 「ゃ」「ゅ」「ょ」「ぁ」등 스테가나는 독립적으로 입력됩니다 (자동 결합이나 조합의 개념 아님)
  • 「ゃ」「ゅ」「ょ」「ぁ」 등 → Shift + 스테가나 선택 / 길게 눌러서 선택
  • 예: Shift + あ → ぁ / Shift + や → ゃ

⌨️ 주요 키맵

  • 히라가나 + 스테가나 기본
  • 카타카나 (전각)
  • 카타카나 (반각)
  • 숫자 및 특수문자
  • 영어 (반각)
  • 특수 스테가나
  • 스테가나 전체
  • 설정 버튼
  • 기본 스테가나 키맵
