JittaKey 사용설명서 (2025.06.29 기준)

✨ 개요: WYSIWYG 일본어 입력의 혁신

JittaKey는 기존의 로마자 변환 방식을 근본적으로 개선한 직관적(WYSIWYG) 일본어 입력 방식을 제공합니다.

  • 입력한 그대로 표시 (What You See Is What You Get)
  • 로마자 베이스 사용자에게도 빠르게 적응 가능
  • 입력 횟수 약 20% 이상 감소
  • 시각적 연상법을 통한 빠른 학습

🔧 핵심 입력 원리

  • 제 1 원칙: WYSIWYG, 즉, 키보드에 나타나는 모든 문자는 1타로 입력된다. 예: あいうえお ぁぃぅぇぉ, わやゆょ ゎゃゅょ, んをｰっヴ, きぎしじちぢにひびぴみり, ゕゖ「」~^_^！、。？
  • 제 2 원칙: 로마자 입력은 ‘い단자음 -きぎしじちぢにひびぴみり’ + 모음 (あうえお)가 연속 입력되는 경우에 적용된다. 예: き + あ = か, き + う = く, き + え = け, き + お = こ
  • 제 3 원칙: ‘い단자음’ + 모음 (あいうえお)의 입력이 필요한 경우에는 ‘い단자음’ 자음을 い 혹은 (sp)로 결정지은 후에, 모음 (あ,い,う,え,お)을 입력한다. 예: きあ를 입력하고 싶은 경우, きいあ로 입력 혹은 き(sp)あ로 입력한다.
  • 제 4 원칙: 요음은 독립입력이다. 이는 로마지의 ‘조합’을 통한 요음 입력과 완전히 다른 부분이다. 예: きょう = き + ょ + う로 보이는 대로 입력한다.
  • 제 5 원칙: JittaKey는 Mobile messenger인 Line, Whatsapp, Kakaotalk 등에서 독보적 효율을 보인다. 특수문자「」~ ^_^ っ ！ 、 。 ？ 는 JittaKey에서 0.2초 꾸욱 누르거나 shift+선택 방식으로 단타로 입력된다. 예: 「今日はいい天気で、よかった」んですね~~^_^！
  • 기타: 카타카나, 숫자, 영어 등 ‘전각, 반각’을 WYSIWYG으로 완벽히 구현하여 모두 보이는 대로의 입력이 가능하다. ※ 아래 주요 키맵 참조

🧠 연상법 기반 키 배치 (Mnemonic Mapping)

키문자연상 힌트
QんQueueing의 “Q”에서 “ing”＝「ん」를 연상할 수 있음
Fを카타카나의「ヲ」와F가 데칼코마니임
Jゆ유도（Juudou）에 “ゆ”음이 포함되어 있음
LよYOLO에서 「よ」음과 L이 포함되어 있음
XーX-mas（크리스마스의 축약형）에 X와 하이픈이 포함되어 있음
Cっ“CHU”의 시작이 C여서, 「っ」를 연상할 수 있음
Vヴ영어의 “V”를 표현한 카타카나임

🈂️ 스테가나 (작은 글자) 입력

  • 「ゃ」「ゅ」「ょ」「ぁ」등 스테가나는 독립적으로 입력됩니다 (자동 결합이나 조합의 개념 아님)
  • 「ゃ」「ゅ」「ょ」「ぁ」 등 → Shift + 스테가나 선택 / 길게 눌러서 선택
  • 예: Shift + あ → ぁ / Shift + や → ゃ

⌨️ 주요 키맵

  • 히라가나 + 스테가나 기본
  • 카타카나 (전각)
  • 카타카나 (반각)
  • 숫자 및 특수문자
  • 영어 (반각)
  • 특수 스테가나
  • 스테가나 전체
  • 설정 버튼
  • 기본 스테가나 키맵
